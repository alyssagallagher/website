Wash 4 All

Student Researcher / Android Developer

Worked on hardware and software used to obtain high resolution water quality data at a price affordable universally. I helped build and design a handheld turbidimeter, using a microcontroller attached to various peripherals including a light sensor to measure and display turbidity, which roughly equates to the amount of particles in a sample of water. I primarily focused on user input and output via an interface controlled using a joystick and displayed on a TFT LCD screen. Features included selecting and starting a measurement type, updating settings, and ultimately displaying measured data. I also programmed an Android application which used the camera to take several sequential images of a cuvette mounted to the device. Image processed thereafter resulted in a numerical measurement value representing the cleanliness of water in a cuvette.

Grassroots Unwired

I work as the lead mobile developer on the Grassroots Unwired Application. My work at Grassroots is an ongoing process where I balance the responsibilities of adding and designing new features, creating a streamlined release process, and improving the existing application architecture. The Android Application's purpose is two fold. First it organizes a canvassers daily schedule by displaying and updating a list of houses that should be visited and surveyed. Next, it pulls dynamic surveys from a backend server that are used to display and collect data during face to face interactions. The application also works in a completely offline state allowing canvassers to continue working without an internet connection, later syncing all local changes when a connection is established.
